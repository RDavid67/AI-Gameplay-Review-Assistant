# Final Demonstration Video

- Local file: `FInal_2mom_home.mp4`
- Duration: approximately 53 seconds
- Final confidence threshold: `0.25`
- Platform: NVIDIA Jetson Orin Nano
- Model: `crowdhuman_visible` SSD-MobileNet / MB1-SSD

The video demonstrates visible-person detection on prerecorded footage. It includes successful detections, crowded multi-person scenes, overlapping detections, and missed detections.

The video file is currently retained locally and is not included in this public repository because it contains identifiable people. Representative detection screenshots are available in the [`results`](../results) folder.

The `FInal` capitalization is retained to match the existing final project file.
